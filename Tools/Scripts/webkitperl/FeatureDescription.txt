#
# This file list all possible WebKit features, it has
# a three columns format:
#
#   COMMAND_LINE_SWITCH  FEATURE_DEFINE  FEATURE_DESCRIPTION
#
# For example, adding the line:
#
# foo-bar  ENABLE_FOOBAR  Toggle Foo Bar support
#
# Will add two new command line switches to build-webkit script,
#
# --foo-bar and --no-foo-bar
#
# And when enabled, ENABLE_FOOBAR will be passed to WebKit, i.e.
# ENABLE(FOOBAR) will evaluate to true.
#
# Each port define a collection of features default to true in the
# file FeatureDefaults{PORT}.txt
#
